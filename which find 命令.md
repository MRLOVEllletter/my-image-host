# which find 命令

## which 命令

用以寻找系统自带的命令的路径

### 语法

which 命令

> eg.
>
> which mv

## find 命令

用以寻找存储的文件/文件夹（搜索）

### 语法

1. find [起始路径] -name “被查找的文件名”（要管理员权限）

   > su - root 切换管理员账号

   这个语法按文件名搜索文件（可与通配符联动）

   > 起始路径表示要从哪里开始搜索

   > 当文件名中带有空格或其他特殊符号，要用" "将包起来

2. find [起始路径] -size +/- n [K/M/G]（要管理员权限）

   这个语法按文件大小搜索文件

   > +/- 代表可选用+或者-，+和-代表大于与小于
   >
   > n代表一个数字
   >
   > K/M/G代表选用单位，可选单位为kb，mb，gb
