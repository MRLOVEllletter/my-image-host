# cd pwd 指令

## cd指令

语法：cd [地址]

> change directory

用途:切换目录

默认情况：直接切换至home目录

## pwd指令

语法：pwd

> print work directory

用途：查看当前目录路径

## 相对和绝对路径

相对路径：以当前目录为起点，无需加/

绝对路径：以根目录为起点，要加/

## 特殊路径符

. 表示当前目录

..表示上一级目录

~表示HOME目录

> cd../..指切换到上两级目录
>
> cd~/Desktop指切换到HOME目录的桌面
