# c语言基本概念

## 1. 写程序的思路六大要求

1. 头文件包含的重要性（用man手册看头文件）

> ## 敢于断错误
>
> **要把gcc的所有报错都解决**
>
> gcc 文件 -Wall

2. 习惯以函数为单位来进行编程

   > 只要能独立成小函数就独立成小函数，可以提高代码的复用性

3. 程序包含声明部分和实现部分

> 变量先定义后使用，声明该放哪里就放哪里

4. return 0;

> echo $?打印出上一条语句的执行结束状态
>
> c的成功为0,失败为非0，来表示各种各样的失败原因，printf返回值是字长

5. 多用空格空行

6. 适当的添加注释

   > 不要让自己忘记有些函数的作用了，并且变量名都最好写完整
